# Iris 的星星泡泡守护者 🌟 (Iris's Star Bubble Guardian)

一个温和、可爱、无暴力元素的 HTML5 Canvas 射击小游戏，专为 10 岁女孩 **Iris** 设计。

游戏以**星星泡泡守护者**为主题，Iris 将控制一只可爱的金色小星星，发射彩色泡泡击碎飘来的淘气小陨石和气球怪，守护梦幻星空的安宁。

---

## 🎮 游戏特色

- **童趣与温馨**：全手绘 Canvas 矢量图形，拥有粉嫩的马卡龙色调和超萌的怪物表情。
- **无暴力真实感**：没有枪支，没有爆炸，也没有流血。击中怪物时，它们会化作五彩缤纷的星星碎屑与泡泡环消散。
- **实时声效合成**：利用 Web Audio API 技术在浏览器中用代码实时合成欢快的泡泡射击音效、得分音效和关卡提升音乐，不依赖任何外部音频文件。
- **支持全平台**：完美适配电脑端（鼠标/键盘）和手机端（手势滑动/触摸），自动适应各种屏幕尺寸。
- **正向激励**：游戏过程中会随机弹出鼓励话语，如 *“Great job, Iris!”*、*“星星能量 +1!”*，失败时也会给予温柔的提示：*“再试一次，你已经很棒了！”*。

---

## 🕹️ 游戏玩法 & 操作指南

### 1. 电脑端 (Desktop)
- **键盘操作**：
  - `A` / `D` 键 或 `←` / `→` 方向键：左右移动金色小星星。
  - `Space` (空格键)：发射粉嫩彩色泡泡。
  - `Esc` 或 `P` 键：暂停/恢复游戏。
- **鼠标操作**：
  - 左右移动鼠标：金色小星星会平滑地跟随鼠标光标移动。
  - 点击左键：向正上方发射彩色泡泡。

### 2. 手机/平板端 (Mobile)
- **手势滑动**：在屏幕上左右拖拽手指，控制小星星移动。
- **屏幕点按**：轻触屏幕任意位置，发射彩色泡泡。

### 3. 游戏机制
- **得分**：每击中一个气球怪或陨石得 **10** 分。
- **升级**：每获得 **100** 分自动进入下一关，怪物下落速度和生成速度会随之小幅提升。
- **生命值**：你有 **5 颗爱心**。当有怪物漏掉飘落到屏幕最下方时，会扣除 1 颗爱心。爱心耗尽则游戏结束。

## 🛠️ 技术实现

- **单文件设计**：整个项目仅包含一个 `index.html`。所有 HTML、CSS 样式、JavaScript 逻辑和音效生成代码均内嵌其中，无需构建工具，可直接在浏览器中打开运行。
- **流畅渲染**：使用 `requestAnimationFrame` 保证游戏以 60FPS 满帧运行，防止屏幕撕裂与闪烁。
- **视网膜适配**：自动计算 `window.devicePixelRatio`，防止在高分屏（Retina 屏）或移动端上Canvas画面模糊，确保极其清晰的矢量效果。

## 🚀 如何运行与部署

### 本地运行
1. 下载本项目文件。
2. 在本地文件夹中找到 [index.html](file://index.html)。
3. 双击 [index.html](file://index.html) 或使用 Chrome/Safari/Edge 等任意现代浏览器打开即可开始守护星空！

### 部署到 GitHub Pages (网页在线玩)
如果你已经将该仓库 push 到了 GitHub：
1. 访问你的 GitHub 仓库页面。
2. 点击 **Settings** (设置) -> 找到左侧菜单的 **Pages**。
3. 在 **Build and deployment** 下的 **Source** 选择 `Deploy from a branch`。
4. 在 **Branch** 栏下拉选择 `main` 分支，目录选择 `/ (root)`，点击 **Save** 保存。
5. 稍等 1-2 分钟，GitHub 会生成一个专属链接（形如 `https://<your-username>.github.io/iris-star-bubble-shooter/`），你可以把这个链接分享给 Iris，在手机或电脑上直接点开即玩！
