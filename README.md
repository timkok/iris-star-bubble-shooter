# Iris 的星星泡泡守护者 🌟 (Iris's Star Bubble Guardian)

一个专为 10 岁女孩 **Iris** 精心改进的、温和且童趣十足的 HTML5 Canvas 射击与收集小游戏。

经过重大体验升级，游戏不仅更耐玩、更精致，还加入了许多适合儿童和无障碍游玩的设计。

---

## 🎮 升级改进特色

### 1. 更丰富好玩的机制
- **连击系统 (Combo)**：连续击中敌人会累积 Combo 连击次数，并在顶部显示可爱的跳动气泡徽章。连击次数越高，得分加成越高！泡泡射空或漏掉怪兽时 Combo 重置，但无惩罚。
- **特殊泡泡道具**：游戏过程中会有三种罕见的特殊气球浮现，射中即可激活特殊魔法：
  - 🌈 **彩虹爆破**：引爆 140px 范围内的所有小怪兽。
  - ✨ **黄金加分**：直接获得大额分数奖励（+50分）。
  - 🛡️ **星光护盾**：赋予玩家 8 秒无敌时间，身披淡粉色护盾光圈，期间漏掉怪兽城堡不扣血。
- **收集小星星**：击败普通小怪有概率掉落亮晶晶的物理**金色小星星**，控制角色去触碰接住，可获得得分加成（并伴有小星星飞入分数栏的飞行动效），还有概率恢复 1 颗爱心！
- **Boss 大挑战**：每逢 5 的倍数关卡（如第 5 关、第 10 关等）会降临一只可爱的**大气球怪 Boss**。它拥有专属的血条并会进行 S 形盘旋，需要射击多次才可消灭。消灭 Boss 会爆出大量金色星星！

### 2. 更多可爱友好的夜空伙伴 (Enemies)
- 🎈 **粉红气球怪**：常规的憨厚反弹小怪物。
- ☄️ **淘气小陨石**：外形圆润、伸着小舌头的紫色顽皮小陨石，下落速度略快。
- ☁️ **瞌睡小乌云**：神情疲惫的浅蓝色乌云，在夜空中缓缓漂浮。
- 🌙 **睡帽小弯月**：戴着粉色睡帽、闭眼假寐的温柔弯月。
- *所有的角色都为无暴力、可爱风格，不含恐怖或真实武器元素。*

### 3. 精致视觉与微动效
- **多层视差背景**：深蓝紫色梦幻夜空 + 闪烁繁星 + 远处缓缓横移的半透明云朵。
- **星星城堡**：屏幕最下方绘制了可爱的城堡剪影（尖顶、小红旗和发出暖光的小窗户）。
- **小星星飞船**：玩家控制的魔法星星增加了尾部喷吐的小气泡火焰，并且胸前多了一个代表 Iris 的粉色字母徽章 **“I”**。
- **动作拖尾 (Motion Trail)**：星星飞船在移动时会留下闪烁的拖尾金沙（可在设置中关闭）。
- **光效与爆破**：消除怪物时拥有更加丰富的气泡和火花散落粒子，升级时会有温馨的白色全屏柔和闪光。

### 4. 更加体贴的儿童友好设置
- **平滑难度梯度**：前两关非常简单，第 3 关起难度开始平缓上升，让 Iris 可以轻松度过前期的上手阶段，顺利挑战到第 5 关的 Boss。
- **失败低挫败感**：失败后没有冷冰冰的 "Game Over"，取而代之的是温馨的 *“守护结束，休息一下”* 提示。
- **内置 3 步新手教学**：首次游戏时会展示一套大字号配图教程（左右移动、射击、消除和收集），点击“跳过”或任意步骤即可直接开始。
- **本地最高分纪录**：使用 `localStorage` 自动记录并显示“最高守护得分”。
- **全方位设置面板 ⚙️**：
  - **声效开关**：一键静音/开启。
  - **游戏难度**：可在 **简单 (Easy)** 和 **普通 (Normal)** 之间自由切换。
  - **特效细节**：支持 **精致 (High)** 和 **节能 (Low)**，节能模式下会关闭拖尾并减少 80% 的粒子动画（可结合 prefers-reduced-motion 系统级无障碍特性）。
  - **重新查看教程**。

## 🕹️ 键盘完整操作

为了满足快捷操控，我们增加了全键盘指令绑定：
- `Enter` 键：在主界面/结束界面直接开始或重新开始。
- `A` / `D` 键 或 `←` / `→` 方向键：左右移动。
- `Space` (空格键)：发射彩色泡泡。
- `Esc` 或 `P` 键：暂停/恢复游戏。
- `R` 键：在暂停时重新开始。
