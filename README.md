# Iris 的星星泡泡守护者

一个温和、可爱、无血腥、无真实暴力感的单文件 HTML5 Canvas 射击小游戏。Iris 控制小星星飞船，用彩色泡泡把调皮小陨石和气球怪轻轻送回星空。

## 如何运行

直接用浏览器打开 `index.html` 即可游玩，不需要安装依赖、不需要构建工具、不需要服务器。

## 玩法

- 左右方向键或 `A` / `D` 移动
- 空格发射彩色泡泡
- 鼠标或触摸：移动到想去的位置，点击或点按发射
- 点击“暂停”可以休息，点击“重来”可以重新开始

## 游戏目标

尽量用泡泡击中飘来的调皮小陨石和气球怪，获得分数、提升等级、收集星星能量。难度会随着分数慢慢增加，但节奏保持简单、明亮、适合儿童。

## 技术说明

- 原生 HTML、CSS、JavaScript
- 单个 `index.html`，所有样式和脚本内嵌
- 使用 Canvas 绘制背景、角色、泡泡、星星和角色
- 使用 `requestAnimationFrame` 运行游戏循环
- 支持桌面端和移动端窗口自适应
