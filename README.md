# Iris 的星星泡泡守护者

一个温和、可爱、鼓励式的 HTML5 Canvas 小游戏。Iris 驾驶星星飞船，发送星星泡泡，收集星星能量，帮助天空小伙伴回到星光里。

在线试玩：
https://timkok.github.io/iris-star-bubble-shooter/

## 玩法说明

- 移动 Iris，避开拥挤的位置。
- 发送星星泡泡，帮助飘来的天空小伙伴。
- 收集星星能量，解锁贴纸和本地成就。
- 每隔几关会出现“星光奖励时间”，节奏更轻松。

## 桌面端操作

- 左右方向键或 `A` / `D`：移动
- 空格：发送星星泡泡
- `P` 或“暂停”按钮：暂停 / 继续

## 移动端操作

- 底部虚拟按钮：左移、右移、发送泡泡
- 也可以在画面左侧拖动移动，在画面右侧按住连续发送泡泡
- 手机竖屏和横屏都可以游玩

## 设置

点击首页“设置”可以调整：

- 音效开关
- 背景音乐开关
- 简单 / 普通 / 挑战 三档难度
- 练习模式
- 减少动画
- 护眼模式
- 重新查看新手引导
- 重置本地数据

设置会保存在本机浏览器的 `localStorage` 中。

## 练习模式

练习模式适合小朋友熟悉操作：

- 不扣爱心
- 不记录最高分
- 下落速度更慢
- HUD 会显示“练习中”

## 每日目标和贴纸

游戏每天会在本机生成一个小目标，例如：

- 今天守护 60 秒
- 今天收集 20 颗星星
- 今天送出 30 个星星泡泡

完成目标或达成本地成就后，可以解锁“我的贴纸”。贴纸不使用外部图片，全部由页面内文字、CSS 或 Canvas 绘制。

## 隐私说明

家长说明：

- 无广告
- 无登录
- 无内购
- 无联网请求
- 分数、设置、每日目标、贴纸和成就只保存在本机浏览器 `localStorage`

## 开发说明

无需安装依赖，也不需要构建工具。

本地运行：

```bash
open index.html
```

或使用任意静态服务器：

```bash
python3 -m http.server 8787
```

然后打开：

```text
http://127.0.0.1:8787/
```
